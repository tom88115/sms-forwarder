# 短信转发助手

这是一个简单的 Android 短信转发应用，用于自动将指定短信转发到您的服务器。

## 主要功能

- **发送者过滤**：只转发来自特定号码的短信（如验证码来源 106*）
- **内容关键词过滤**：只转发包含特定关键词的短信（如"拼多多"）
- **Webhook 转发**：自动将短信内容以 JSON 格式 POST 到您的服务器
- **后台运行**：开机自动启动，持续监听短信

## 使用场景

主要用于拼多多店铺登录验证码的自动转发：
1. 在闲置安卓手机上安装此 APP
2. 配置转发规则：内容关键词设为"拼多多"
3. 配置您的服务器接收地址
4. 当收到拼多多验证码短信时，自动转发到服务器

## 快速配置

1. 打开应用，点击右下角 + 按钮添加规则
2. **发送者**：输入 `*` 表示接收所有短信，或输入 `106` 只接收验证码
3. **内容关键词**：输入 `拼多多`（只转发包含此关键词的短信）
4. **服务器地址**：输入您的接收 API 地址

## 推送数据格式

收到符合条件的短信后，会向服务器发送如下 JSON：

```json
{
  "from": "1069xxxxx",
  "text": "【拼多多】验证码：123456，5分钟内有效。",
  "sentStamp": 1234567890000,
  "receivedStamp": 1234567890123,
  "sim": "sim1"
}
```

## 权限说明

应用需要以下权限：
- **接收短信**：用于监听收到的短信
- **网络访问**：用于将短信转发到服务器
- **开机自启**：确保服务持续运行
